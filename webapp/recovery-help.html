Database Recovery/Restoration

  • Using the “.sql” file from the backup feature of the database, the user can restore the database to its previous state. This is useful in cases when the database has been corrupted or when data integrity has been compromised.
  • The “Database Recovery Tool” which is installed along with the system is used in Database Recovery/Restoration. First, input the username and password for the database.  The default username is “root”.
  • Press the “Start Recovery” button. The notification “Login Verified. Connection Successful” should now visible. Press “OK”. A “Warning Message” will now be visible. Read the message carefully. If you want to proceed, press “OK”. A File Chooser will now be visible. Navigate to the location of the “.sql” file. Once the file has been selected, press “Select Database Files (.sql)”. Database Recovery will now start. This may take a couple of minutes. Wait until the notification ”Database Restoration Complete” is visible. Close the Database Recovery Tool.
