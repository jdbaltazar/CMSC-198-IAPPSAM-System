Post Installation Procedure

  • In adding and/or updating entries keep in mind that highlighted fields are required. Thus, they must be properly filled out. Notice that default values have also been provided.
  • In the main menu page, click the “Stock Properties” button. Once in the “Stock Properties” page, add as many entries as necessary because these will be used in adding items and some forms. Entries for each type in “Stock Properties” must be unique otherwise they will not be saved.
  • Next, add the items.  When item is successfully added, the user is navigated to the “List Items” page where all items can be viewed. Otherwise, “Add Item” page is returned.
  • Return to the main menu. In the settings section, add Building entries.
  • Similarly, add Supplier entries. Notice that in the “Add Supplier” page, there is the link “Add Supplier to Existing Contact Person” option. This option is to be used when two or more establishments (i.e. Supplier) have the same contact person.
  • Return to main menu and proceed to the “Add Division” page. Once a division is added, the user is navigated to the “Division List” page. By clicking the “View” button, the user can choose to add an Office to the selected division by clicking the “Add Office” button.
  • Add Employee entries. At least one “Designation” and “Division/Office” pair must be filled up to save an employee.
  • Once finished adding entries for the “Settings” section, the user can opt to add more accounts. Notice that there is the link “Create Account For Existing Employee”. This option is to be used when the user wants to create an account for an employee that is already in the database.
