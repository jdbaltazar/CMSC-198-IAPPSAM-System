Database Modifications

  • In the “Database” section, the user can view the changes (in order of the most recent) made by the accounts with corresponding date and time of the activity in the format "MM-dd-yyyy [HH:mm:ss]". A short description of the activity along with the username of the account used.
  • It is strongly advised that the user back-up the data in the database from time to time. To do so, the user must click the “Backup” button in the main menu and save the “.sql” file in a desired location in his/her computer.The file name is in the format “MM-dd-yyyy [HH:mm:ss]”.
