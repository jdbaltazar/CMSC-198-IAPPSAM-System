Filling-up Forms

  • Each form must be properly filled out otherwise it will not be saved.  At least one item must be added. Highlighted fields are required.
  • All fields requiring numerical input only accept non-negative numbers. All forms except for Annual Procurement Plan require numerical input that is greater than zero.
  • Click the “SUBMIT Form” button in order to save the form. The user will then be navigated to the “View Form” page where the form entries can be seen.
  • From there, the user can opt to export the form in either PDF or Excel format by clicking the corresponding links below the “Logout” button.
